Ikat Generator

The Ikat Generator is an interactive web tool that facilitates generating Ikat patterns with unique moments and sensibilities using the various controls. This allows for the creation of unique design choices and finer control over the output.

This project was created by Meilin and Salil. during the DigiKat Hackathon. See other projects from the hackathon here!

Select a pattern in the drop down menu to start playing around! If you'd like to use your own patterns, trying using this version.
